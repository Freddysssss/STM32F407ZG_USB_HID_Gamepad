# STM32F407ZG_USB_HID_Gamepad

实验目的：
利用USB外设实现游戏手柄的简易功能，通过USB将设备与电脑连接，简单控制小游戏。本次实验采用5个按键分别控制游戏手柄的X键、Y键、A键、B键、右侧摇杆键，使用两路ADC检测右侧摇杆的x坐标和y坐标，进而控制前后左右。
